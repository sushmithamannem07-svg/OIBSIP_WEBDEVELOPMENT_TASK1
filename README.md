# AdSpot — Modern Marketing Landing Page

AdSpot is a modern, responsive landing page built with HTML and CSS.  
It is designed for marketing agencies, ad platforms, or startups to showcase services, display advertisements, and convert visitors effectively.

---

## Features

- Clean and responsive design  
- Fast loading and mobile-friendly  
- Built with semantic HTML5 and modern CSS  
- Includes sections for:
  - Hero / Headline
  - Features
  - Pricing
  - Call-to-Action
  - Contact Form  
- Ideal for ads, marketing campaigns, or small business landing pages

---

## Tech Stack

- HTML5  
- CSS3 (Flexbox and Grid)  
- Google Fonts (Inter)  
- Fully static (no dependencies or frameworks)
